# Changelog

All notable changes to this project will be documented here.

## 0.1.0 - 2026-05-25

Initial public release.

### Added

- Minecraft chat to Discord.
- Discord chat to Minecraft.
- Optional webhook mode for player-style chat output.
- Minecraft and Discord account linking.
- Discord commands for linking, unlinking, online count, and player list.
- Permission-based Discord role sync.
- Styled Discord events for joins, first joins, quits, advancements, deaths, lifecycle, status, and broadcasts.
- Configurable channel routing.
- GitHub Actions build workflow and release jar.
