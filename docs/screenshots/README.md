# Screenshot Guide

Use this folder for GitHub, marketplace, and documentation screenshots.

## Recommended Captures

- `chat-relay.png`: Minecraft chat relayed into Discord.
- `discord-to-minecraft.png`: Discord chat appearing in Minecraft.
- `join-event.png`: a styled join or first-join embed.
- `advancement-event.png`: a styled advancement embed.
- `role-sync.png`: sanitized role sync status or linked account status.

## Capture Notes

- Redact bot tokens, webhook URLs, guild IDs, channel IDs, and private channels.
- Use a Discord channel with non-sensitive test messages.
- Keep embeds visually clean and cropped around the relevant message.
- Keep images at least 1280px wide when possible.
